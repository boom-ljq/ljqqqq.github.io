Java 接口与lambda表达式

Posted on 2018-10-22 Edited on 2020-02-02 In Coding

本章要点：

  • 接口指定了一组实现类必须提供的方法。
  • 接口是任何实现该接口的父类。因此，可以叫类的实例赋值给接口类型的变量。
  • 接口可以包含静态方法。接口的所有变量默认是static和final的.
  • 接口可以包含默认方法，实现类可以继承或者覆盖该默认方法。
  • lambda表达式代表一段代码块，可以在稍后的某个时间点执行。类可以嵌套在其他类中。
  • lambda表达式转变为函数式接口。
  • 方法引用和构造函数引用在不调用的他们的情况下引用方法或构造函数。
  • lanmbda表达式和局部内部类可以有效地访问来自闭合作用域的final变量。

Java 异常 断言和日志处理

Posted on 2018-10-15 Edited on 2020-02-02 In Coding

本章要点：

  • 当你抛出了一个异常时，控制权转移到最近的异常处理器
  • 在Java中，由编译器跟踪已检查异常 checked exception。
  • 使用 try/catch 结构处理异常。
  • 在正常执行完之后或者当发生异常时，try-with-resources 语句会自动关闭资源。
  • 使用 try/finally 结构处理那些不论执行是否正常地进行都要发生的行为。
  • 你可以捕获和重新抛出一个异常，或者将它与另外一个异常链起来。
  • 堆栈踪迹step trace信息描述了一个程序执行中挂起时所有方法的调用情况。
  • 一个断言检查一个条件，如果启用了断言检查，当条件不满足时，会抛出一个错误。
  • 日志处理器按照层次来分类，他们可以接收从 SERVER 到FINEST` 级别范围的日志信息。
  • 日志处理器可以将日志信息发送给代替目的地，格式化器 formatter 控制信息格式。
  • 你可以通过日志配置文件控制日志属性。

Java 继承与反射

Posted on 2018-10-15 Edited on 2020-02-02 In Coding

本章要点:

  • 子类可以继承或者覆盖父类的方法。
  • 使用关键字 super 来调用父类方法或者构造函数。
  • final 方法 不可能被覆盖，final 类 不能被继承。
  • abstract 方法 没有实现，abstract 类 不能被实例化。
  • 子类方法可以访问子类的受保护成员，但仅适用于同一个子类的对象。
  • 所有类都是 Object 类 的子类，它提供了 toString、equals、hasCode 和 clone 方法。
  • 所有枚举类型都是 Enum 类 的子类，它提供了 toString、valueOf 和 compareTo 方法。
