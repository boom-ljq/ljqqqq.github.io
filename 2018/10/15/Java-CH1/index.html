基本类型

最简单的Java类型被称为基本类型。其中四种是整型，两种是浮点类型，另外一种是用在字符串编码中的字符类型 char，还有一种是表示真假值的 布尔类型 boolean。

整型

整型是没有小数部分的数字。允许负值。这四种整型分别是 int、long、short、byte。

  • int 4个字节
    （刚好超过20亿）
  • long 8个字节
    这个范围贼大。如果 long类型 还不够用，则使用 BigInteger类（这不是 基本类型）。
  • short 2个字节
    -32,768 ~ 32,767
  • byte 1个字节
    -128 ~ 127

常量 Integer.MIN_VALUE 和 Integer.MAX_VALUE 分别表示最小和最大的整数。同样的，Long、Short、Byte 也有 MIN_VALUE 和 MAX_VALUE 常量。

使用 long 变量值时带上 L 后缀（例如，400000000L），byte 或者 short 变量值在使用时没有这种语法。采用强制转换*（例如，(byte)127）。

十六进制数值使用 0x 前缀（例如，0xCAFEBABE），二进制 数值使用 0b 前缀（例如，0b1001 就是9），
八进制 使用 0 作为前缀（例如，011 是9），这会令人产生误解，最好不使用。

你可以给数字添加下划线。例如 1_000_000，0b1111_0100_0010_0100_0000 都可以表示100万。添加下划线让阅读更加容易，Java编译器会直接删除它们。

Java中的 整型 是有符号的。但是如果你使用的值不可能是 负数，并且实际需要额外一个比特，那么你可以使用 将数值翻译成无符号类型 的方法。例如，将范围为-128127的 byte 类型值用作0255的值，并且进行一些源于二进制运算的操作，例如加法和减法。对于其他操作，可以调用 Byte.toUnsignedInt(b)方法，可以获得0~255的整数值。

浮点类型

浮点类型是指由分数的数字。允许负值。这两种 浮点类型 分别是 float 和 double。

  • float 4个字节
    6~7位有效的小数位数
  • double 8个字节
    15位有效的小数位数。现在默认使用 double 精度的数字作为 浮点类型 的数值。

一些特殊的 浮点值 有: Double.POSITIVE_INFINITY 代表 正无穷大，Double.NEGATIVE_INFINITY 代表 负无穷大，DOUBLE.NaN 代表 非数值。
（例如，1.0 / 0.0 的结果就是 正无穷大，0.0 / 0.0 或者 负数的平方根 就会产生 非数值 NaN）
所有 非数值 都被认为是彼此不同的。因此你必须调用 Double.isNan() 来检查数值是否为 NaN。 你也可以用方法 Double.isInfinite 来检查数值是否为 正负无穷大.

float类型 有个 F 后缀（例如，3.14F）。没有 F 后缀的 浮点类型 默认为 double类型。你可以选择使用 D 作为 double类型 的后缀（例如，3.14D）。

浮点数 不适合做金融运算，因为有些 浮点数 在计算机中没有精确的二进制表示（例如 9/10 打印结果为 0.8999999999999999，并非0.9）。如果你需要精确而无舍入误差的数字计算，则可以使用 BigDecimal类。

char型

char型 描述了Java使用的UTF-16字符编码中的”编码单元”。你可以不会经常使用 char类型。编码单元可以用十六进制表示，使用前缀\u。例如，\u004A和J相同。
特殊编码\n、\r、\t、\b、代表换行、回车、tab和退格。反斜杠 \ 作为转义符。

布尔型

布尔型有两种值。true 和 false。
在 Java 中，布尔类型不是数字类型，布尔值和整数0和1没有关系。

变量

Java是强类型语言。 变量值只能是某个具体类型的值。当你声明变量时，需要指定变量的类型，名称及一个可选的初始值。

字符串

字符串连接

使用操作符+可以将两个字符串连接起来。

String firstName = “LI”;
String fullName = firstName + “KAKI”;

你也可以将字符串和另一个值连接在一起，该值转换为字符串。

int age = 23;
String output = age + “years”;

使用join()方法可以将几个以分隔符隔开的字符连接起来.

String names = String.join(“, “, “Peter”, “Amy”, “Lee”);

字符串拆开

要将字符串拆开，你可以使用 subString() 方法。例如:

String greeting = “Hello World”;
String w = greeting.subString(7,12); //w现在为 “World”

如果你想用分隔符来将字符串中的所有子字符串提取出来，可以使用 split() 方法，分隔符可以是 正则表达式。

String names = “Peter, Amy, Lee”;
String result = names.split(“, “);

字符串比较

请不要用 == 来比较字符串，而是使用 equal() 方法。

数字与字符串转换

将整数转换为字符串，可以调用静态方法 Integer.toString()

int n = 10;
String str = Integer.toString(n);

该方法也可以支持两个参数，可以指定基数，范围2~36。

str = Integer.toString(n, 2); //str现在为”1010”

有一种更加简单的方法，但是效率稍低。

str = “” + n;

相反地，将包含整数的字符串转换成数字，使用 Integer.parseInt() 方法:

String str = “1010”;
n = Integer.parseInt(str); //n现在为”1010”

这个方法也支持两个参数

n = Integer.parseInt(str, 2); //n现在为”10”

对于 浮点数，可以使用Double.toString() 方法和 Double.parseDouble() 方法

输入与输出

调用 System.out.println() 时，输出发送到”标准输出流”，然后再窗口中显示出来。但是从”标准输入流”读取并不是那么简单，因为相应的 System.in 对象只能读取单个字节，要读取字符串和数字，构造一个依附到System.in的Scanner。

Scanner in = new Scanner(System.in);

nextLine() 方法从输入读取一行。

String input = in.nextLine(); //next()方法会过滤空格，enter，tab，并且其是为结束符。
int age = in.nextInt();
double salary = in.nextDouble();

使用hasNextLine()、hasNext()、hasNextInt()、hasNextDouble() 可以检查时候有另外一个行，另外一个单词，整数或者浮点数可以读取。

Scanner类 位于java.util包中。要使用该类，可以在文件中引用。

import java.util.Scanner

格式化输出

以%字符开始的每一个格式说明符都会被相应的参数替代。转换字符 说明了要被格式化的值所属的类型: f 是浮点数字，s 是字符串，d 是十进制整数。

System.out.printf(“%8.2f”, 1000.0 / 3.0); //宽度为 8，小数点后精度为 2，类型为浮点数。

数组和数组列表

数组是手机相同类型的多个元素的基本编程结构。Java内置了数组类型，并且也提供了根据需求随时增长或缩减的ArrayList类。

使用数组

构造数组

使用new操作符去构造数组时，用默认值填充数组。
数字类型（包括char类型）的数组用0填充。
Boolean类型的数组用false填充。
对象数组用空引用填充。

数组列表

数组一旦构造后，数组的长度就不能被改变。为了解决这个问题，可以使用ArrayList类。
数组列表和数组的语法完全不同，数组列表是一个类，你可以使用构造实例和调用方法的语法。
